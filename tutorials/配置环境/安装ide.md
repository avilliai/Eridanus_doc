# 安装ide

【Eridanus项目】和【Eridanus sdk】不是一个东西。
- 【Eridanus项目】即主仓库，已经完成了诸多功能。
- 【Eridanus sdk】是为其他开发者提供的开发工具，使你能够另起炉灶编写bot源码
